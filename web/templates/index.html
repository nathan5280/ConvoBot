Overview


Provide navigational robot control in the real world using a convolutional neural network trained in a simulated virtual enviornment.

Simulated environment was rendered in Blender a leading open source animation studio. 120,000 64x64 pixel RGB images were rendered of the simple 3 block enviornment to train the CNN.

The Robot is a mBot Ranger. The mBot Ranger uses a Arduino Mega 2560 micro controller. This is suplemented with a Raspberry Pi 3 and Raspberry Camera.
